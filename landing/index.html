autoyield

Non-custodial RWA + yield service for AI agents on Solana.

Solana mainnet MCP-native Funds in your wallet

Connect this MCP to Claude, Cursor, or any MCP-compatible AI. Your AI can then compare on-chain yields, quote tokenized US equities, and build ready-to-sign transactions. You sign every action in your own Phantom wallet — we never custody.

What it does

  • Compare USDC lending APY across Solana / Ethereum / Base / Arbitrum
  • Buy / sell Backed xStocks (NVDAx, AAPLx, TSLAx, SPYx) via Jupiter
  • Deposit USDC into Ondo USDY tokenized treasuries (~4-5% APY)
  • Snapshot any wallet's portfolio with live USD valuations

Install

Add to ~/Library/Application Support/Claude/claude_desktop_config.json:

{
  "mcpServers": {
    "autoyield": {
      "command": "npx",
      "args": ["-y", "@kaditang/agent-wallet-mcp"]
    }
  }
}

Why this exists

Tokenized US treasuries (Ondo USDY, BlackRock BUIDL) and tokenized equities (Backed xStocks) became the fastest-growing RWA category in 2026. After the GENIUS / CLARITY Acts restricted stablecoin issuer interest, the cleanest yield path is direct ownership of these securities — but the infrastructure has been built for traders, not AI agents. autoyield wraps the boring-but-correct execution in a service your AI can drive.
